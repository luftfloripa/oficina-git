Oficina de GitHub com Juh Achmann

A aluna Juliana Achmann, aluna do IFSC que fez intercâmbio para Portugal com bolsa, está fazendo uma oficina de GiHub com os alunos do Curso Técnico Desenvlvimento de Sistemas do segundo semestre de 2024.
A oficina consiste em aprender como baixar o Git e GitHub para desktop, criar a conta, os primeiros passos de como criar repositório, branch, como fazer um FORK, como clonar, adicionar colaboradores no projeto, como fazer o pull dos arquivos no github.
Da mesma forma, de fazer o commit, e depois o push.
Bela aula, baita aprendizado.
